# App Integration Examples

Generic, technology-agnostic examples referenced from
[`docs/app-integration-patterns.md`](../../docs/app-integration-patterns.md). These are examples
to copy and adapt, not configuration this repo applies automatically.

- [`generic-oidc-client.md`](generic-oidc-client.md) — Pattern 1: native OIDC app integration.
- [`generic-forward-auth.md`](generic-forward-auth.md) — Pattern 2: reverse-proxy forward-auth.

No specific app (e.g. `photos.example.com`, `tools.example.com`) is configured here — actual per-app
onboarding happens in Phase 4/5, following the checklist in
[`docs/oidc-integration.md`](../../docs/oidc-integration.md) or
[`docs/proxy-auth-integration.md`](../../docs/proxy-auth-integration.md).
